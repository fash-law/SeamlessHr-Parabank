# SeamlessHr-Parabank
This repository is for SeamlessHr Assessment


## Prerequisites

- Node.js (https://nodejs.org/)
- npm (https://www.npmjs.com/)
- git bash

## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/fash-law/SeamlessHr-Parabank.git
    cd your-repository
    ```

2. Install dependencies:

    ```bash
    npm install
    ```
